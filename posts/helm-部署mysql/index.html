Helm 部署MySQL

 2022.4.28  2022.5.5  Kubernetes  5583  12 分钟

前面几天煞费苦心的在Ubuntu上部署了kubernetes, 具体怎么部署的请出门右转到 Linux常用工具使用

Kubernets集群有了，就得在Kubernets上部署点什么东西是吧。MySQL作为一名合格的Backend，应该都知道。所以就尝试部署一下MySQL。

有使用Kubernets经验的人都知道，Kubernets有很多的yaml文件需要编写，deployment，service等yaml文件。还是比较繁琐的。所以出现了Helm这个工具。helm这个其实没什么高级的，就是将deployment，service等yaml文件抽象成模版，在这些模版中使用变量，使用helm可以编辑这些变量这样就能很快的在kubernets中进行部署了。

在继续之前，最好了解一下Helm，kubernetes存储这些知识点

首先添加repo

$ helm repo add bitnami https://charts.bitnami.com/bitnami

$ helm repo update              # Make sure we get the latest list of charts

查看chart中有哪些变量
